# SeniorCare SOS App

An Android application designed to provide quick emergency assistance for senior citizens.

## 🚀 Features
- One-tap SOS alert
- Sends SMS with live location
- Emergency calling feature
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Java
- XML
- Android Studio

## 📱 How It Works
1. User presses SOS button
2. App sends SMS with location
3. Automatically calls emergency contact

## 🔮 Future Improvements
- Real-time location tracking
- Voice activation
- Notification alerts
